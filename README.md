# PolyHouse - White-Label Prediction Market Platform

Launch your own branded prediction market in minutes using Polymarket's infrastructure.

## For Operators

PolyHouse is a turnkey white-label solution for launching prediction markets. We handle all the technical complexity - you focus on your brand and community.

### What's Included

- **Prediction Markets** - Full Polymarket integration with real liquidity
- **Leaderboard System** - Track top traders and create competition
- **Referral Program** - 4 configurable strategies to drive growth
- **Points & Rewards** - Gamification to increase engagement
- **Fee Collection** - Multi-wallet revenue distribution
- **Admin Panel** - Configure everything without touching code
- **Theme System** - Complete white-label branding
- **Mobile Optimized** - Beautiful UI on any device

## Quick Start for Operators

### 1. Deployment
We handle all deployment for you:
- Replit hosting (includes database)
- Custom domain configuration
- SSL certificates
- Ongoing maintenance

### 2. Configuration
Access your admin panel at `yourdomain.com/admin` and configure:
- **Branding:** Logo, colors, site name
- **Theme:** Dark mode, accent colors, fonts
- **Fees:** Revenue wallets, split percentages
- **Referrals:** Growth strategies, leaderboard settings
- **Points:** Reward rules, redemption options

### 3. Launch
Your branded prediction market is live!

## Core Features

### Prediction Markets
- Real Polymarket contracts with actual liquidity
- Sports, politics, crypto, entertainment markets
- Mobile-friendly betting interface
- Live odds updates
- BetSlip with fee transparency

### Leaderboard & Referrals
- Multiple referral strategies:
  - Growth Multiplier (viral growth)
  - Revenue Share (passive income)
  - Milestone Quest (gamification)
  - Team Volume (community building)

- Historical archives
- Social sharing

### Admin Panel
Configure everything without code:
- White-label branding
- Fee collection wallets
- Referral system rules
- Points system rewards
- Theme customization
- User management

## Technical Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (Replit managed)
- **Blockchain:** Polygon (Polymarket integration)
- **Wallet:** Privy + Safe (automated trading)
- **Deployment:** Replit (automated)
